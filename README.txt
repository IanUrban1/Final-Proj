To run the MichaeK_IanU_Final.sprj file on your computer:
1. Download and open the SHADERed folder from git hub
2. Move the MichaeK_IanU_Final.sprj to SHADERed\saves
3. Move the finalProject folder to SHADERed\saves\shaders
4. Open the MichaeK_IanU_Final.sprj file. This should launch SHADERed with all the necessary stuff
